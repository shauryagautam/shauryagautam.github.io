# Premium Product Manager Portfolio

A high-impact, FAANG-level personal portfolio designed for Product Managers, Founders, and Operators. Built with **Astro**, **React**, and **Tailwind CSS**.

## 💎 Design Philosophy
- **Minimalism**: Inspired by Apple and Tesla. Clean typography, high whitespace, and clarity over clutter.
- **Storytelling**: Focused on the "Why" and "How" of product building, not just the "What."
- **LiquidGlass System**: Subtle animated background orbs and glassmorphic UI elements for a premium feel.

## 🏗️ The Three Pillars Architecture
This portfolio is built to showcase a multi-domain expertise:
1. **Shaurya&Company**: A product & growth consulting studio.
2. **Astra**: A fullstack Golang framework for developers.
3. **Aarivu**: A consumer social platform venture.

## 🛠️ Tech Stack
- **Framework**: [Astro](https://astro.build/) (Static-first, Island architecture)
- **UI Logic**: [React](https://reactjs.org/) (Hydrated only where necessary)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```

## 📦 Deployment (GitHub Pages)

1. Update the `site` and `base` fields in `astro.config.mjs`.
2. Push your code to a GitHub repository.
3. Go to **Settings > Pages** in your repo.
4. Select **GitHub Actions** as the source.
5. Use the standard Astro deployment workflow.

## 📝 Customization
- **Content**: Edit `src/pages/index.astro` and components in `src/components/`.
- **Styling**: Modify the `@theme` block in `src/styles/globals.css`.
- **Case Studies**: Update the Notion URLs in `src/pages/index.astro`.
